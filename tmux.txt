tmux = start tmux

^b = before any command

^b c = for new terminal

^b n = for next terminal

^b w = to show a tree list of terminals

^b  "" = split the terminal in 2 horizontal panes

^b L/R or U/D arrow = change between panes

^b % = splits the terminal vertical

^b q = flash the number of each pane

^b x = close the curent pane

^b 0-9 = move to a specific pane

^b o = cycle between panes
